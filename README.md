# ⭐ 星空跳跳問答：熊抱青春記

一個以 Three.js 製作的 3D 中文問答跳躍遊戲。玩家控制小紅熊貓跳上正確答案星星，完成 10 題後到達天空書之城堡。

## 遊戲功能

- 10 條《熊抱青春記》廣東話故事問題
- 3D 小紅熊貓角色、星空、月亮、雲層及書之城堡
- 5 條生命及答錯／墮下判定
- 隨機法寶：翅膀、巨人、小貓二段跳
- 電腦鍵盤及手機／平板觸控操作
- 完成後可透過電郵傳送成績

## 操作

### 電腦

- `← → ↑ ↓`：移動
- `空白鍵`：跳躍
- 滑鼠點擊畫面或按 `P`：啟動隨機法寶

### 手機／平板

使用畫面左下方方向鍵，以及右下方的「跳躍」和「法寶」按鈕。

## 修改問題

問題及答案放在 `scene.js` 的 `QUESTIONS` 陣列：

```js
{ q: "問題", a: ["答案 A", "答案 B", "答案 C"], c: 0 }
```

`c` 代表正確答案的位置：`0`、`1` 或 `2`。

## 檔案

- `index.html`：遊戲頁面
- `style.css`：畫面及手機控制介面
- `scene.js`：題目、天空及燈光
- `world.js`：角色、平台、城堡及粒子效果
- `game.js`：遊戲狀態、答案及生命邏輯
- `main.js`：物理、碰撞、動畫及主循環
